Research Interest

I am currently working on enhancing machine learning techniques with semantic data for market segmentation, semantic information extraction, and user profiling. In the past I worked on Ontology Based Data Access (OBDA), Knowledge Representation and Reasoning (KRR) and Natural Language Processing (NLP).

In Ontology Based Data Access (OBDA), the objective is to access data trough a conceptual layer. Usually, this conceptual layer is expressed in the form of an OWL or RDFS ontology, and the data is stored in relational databases. The terms in the conceptual layer are mapped to the data layer using mappings. I have intensively worked on query rewriting techniques for OBDA. That is, answer the queries posed by the user (e.g., SPARQL queries) by translating them into queries over the database (e.g., SQL).

In my Ph.D. dissertation, I worked modeling and reasoning over domains that change over time using Transaction Logic (TR). I extended TR to model and reason with recursive, non-monotonic, and complex actions. I have also worked providing logical reconstructions of the semantics of Production systems in Mu-Calculus, Fix Point logic, and Transaction Logic. Along the same line, I also provided an operational semantics and a model-theoretic semantics for the combination of production systems with description logic ontologies.

Regarding NLP, I have worked on natural language generation, and a framework that allows processing Korean text, efficiently producing RDF triples, and querying the outcome using SPARQL. The final goal was to chain several Korean NLP tools and be able to efficiently publish the outcome on the web linking Korean text entities with Wikipedia/DBpedia.
